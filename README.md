## 开发

* 安装依赖

  ``` bash
  yarn
  ```

* 启动开发环境

  ``` bash
  yarn dev
  ```
  访问: http://localhost:3009


* 本地 API 代理
  
  可以在这里修改[配置](./proxy.config.json)来实现本地 API 访问

* 包体积优化

  ``` bash
  yarn analyze
  ```
